## 🤔 Como rodar o código?

### Pré-requisitos

- Node.js (v16 ou superior)
- Mysql

### Passos para Configuração

1. **Clone o repositório:**

   ```bash
   git clone https://github.com/Batissta/idea-keeper.git
   cd idea-keeper
   ```

2. **Instale as dependências:**

   ```bash
   npm install
   ```
